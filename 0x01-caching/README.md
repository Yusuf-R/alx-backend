# Caching System

This project aims to teach you about caching systems and their limitations. By the end of this project, you will be able to explain the following concepts without the help of Google:

- What a caching system is
- What FIFO means
- What LIFO means
- What LRU means
- What MRU means
- What LFU means
- What the purpose of a caching system is
- What limits a caching system has

## Learning Objectives

At the end of this project, you are expected to be able to explain to anyone, without the help of Google:

- What a caching system is
- What FIFO means
- What LIFO means
- What LRU means
- What MRU means
- What LFU means
- What the purpose of a caching system is
- What limits a caching system has
